## 📚 Complete search guide (English)

All commands are supported. You can send a single query or multiple lines (one request per line) – they will be executed simultaneously.

### 📧 Email search

| Example                          | Description                                    |
|----------------------------------|------------------------------------------------|
| `example@gmail.com`              | Search for a specific email address           |
| `example@`                       | Search without domain (only local part)       |
| `@gmail.com`                     | Search for a specific email domain            |

### 👤 Name / nickname search

| Example                         | Description                       |
|---------------------------------|-----------------------------------|
| `Petrov`                        | Search by surname                 |
| `Petrov Maxim`                  | First + last name                 |
| `Petrov Sergeevich`             | Surname + patronymic              |
| `Maxim Sergeevich`              | First name + patronymic           |
| `Petrov Maxim Sergeevich`       | Full name                         |
| `ShadowPlayer228`               | Nickname                          |

### 📱 Phone number search

| Example             | Description                         |
|---------------------|-------------------------------------|
| `+79002206090`      | International format                |
| `79002206090`       | National format (without +)         |
| `89002206090`       | Domestic format (with leading 8)    |

### 🔑 Password search

| Example   | Description        |
|-----------|--------------------|
| `123qwe`  | Search for a password (leaks/dumps) |

### 🚗 Vehicle search (Russian databases)

| Example                 | Description                              |
|-------------------------|------------------------------------------|
| `O999МУ777`             | Search by Russian license plate          |
| `ВО4561АХ`              | Search by traffic police code            |
| `XTA21150053965897`     | Search by VIN (Vehicle Identification Number) |

### ✈ Telegram account search

| Example          | Description                        |
|------------------|------------------------------------|
| `Petrov Ivan`    | Search by first and last name      |
| `314159265`      | Search by Telegram user ID         |
| `Petivan`        | Search by Telegram username        |

### 📘 Facebook account search

| Example          | Description                |
|------------------|----------------------------|
| `Petrov Ivan`    | Search by name             |
| `314159265`      | Search by Facebook user ID |

### 🌟 VKontakte account search

| Example          | Description                |
|------------------|----------------------------|
| `Petrov Ivan`    | Search by name             |
| `314159265`      | Search by VK user ID       |

### 📸 Instagram account search

| Example          | Description                |
|------------------|----------------------------|
| `Petrov Ivan`    | Search by name             |
| `314159265`      | Search by Instagram user ID |

### 📍 IP address search

| Example      | Description        |
|--------------|--------------------|
| `127.0.0.1`  | Search by IPv4 address |

### 📃 Mass search from a file

- File encoding: **UTF‑8**
- One request per line
- Upload the file to the bot – all queries will be processed in bulk.

### 🔄 Composite queries (mixed data types)

You can combine different types of data in a single line. The bot will automatically understand the context.

| Example                                               |
|-------------------------------------------------------|
| `Petrov 79002206090`                                  |
| `Maxim Sergeevich 127.0.0.1`                          |
| `Petrov Maxim Sergeevich 16/02/1995` (date of birth)  |
| `ShadowPlayer228 example@gmail.com`                   |
| `Maxim Sergeevich Moscow` (location)                  |
| `example@gmail.com 123qwe`                            |
| `ShadowPlayer228 16.08.1994`                          |

### 🧠 Multiple simultaneous searches

Simply write **one request per line** and send them all in one message. The bot will execute all queries in parallel.
